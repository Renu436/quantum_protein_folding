🧬 Hybrid Quantum-Classical 3D Protein Folding Using HP Model and QAOA

A research-oriented project that combines Quantum Computing and Classical Optimization to solve the 3D Protein Folding Problem using the Hydrophobic-Polar (HP) Model and the Quantum Approximate Optimization Algorithm (QAOA).

🚀 Features

* Hybrid Quantum-Classical optimization
* 3D Protein Folding using the HP Model
* QAOA implementation with Qiskit
* Classical optimization for comparison
* Interactive visualization of folded proteins
* FastAPI backend
* React frontend
* Performance analysis and energy comparison
* Modular and scalable architecture

⚙️ Installation

Clone the repository:

git clone https://github.com/Renu436/quantum_protein_folding.git
cd quantum_protein_folding

Create a virtual environment:

python -m venv venv

Activate it:

Windows

venv\Scripts\activate

macOS/Linux

source venv/bin/activate

Install dependencies:

pip install -r requirements.txt

Run the application:

python main.py
